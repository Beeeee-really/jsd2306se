## MySQL

### 登录与登出

- **登录远端服务器** mysql -u(用户名) -h(ip地址) -P(端口号) -p(密码)
- **登陆本地服务器** mysql -u(用户名) -p(密码)
- **登出** exit *(ctrl+c)*

## SQL

- 编程语言
- SQL是Structured Query Language的缩写，翻译成中文为结构化查询语言
- SQL是与数据库进行交互的一种标准，但是不同数据库会有基于SQL的一些不同实现

### SQL的用处

- 创建数据库中的对象
- 访问数据库中的对象
- 更新数据库中的数据

### SQL类型

- 不同类型的SQL语法不同
- 基于业务选择不同的SQL进行应用
